Speed Test with Encryption

Select one of the options described below and choose a packet size and click [Test].

  • Guest: encrypts only once with AES-256.
  • Basic: encrypts twice, using AES-256, AES-CTR.
  • Advanced: encrypts three times, using AES-256, AES-CTR, HMAC-SHA256.
  • Admin: encrypts four times, using AES-256, AES-CTR, HMAC-SHA256, XChacha20.
